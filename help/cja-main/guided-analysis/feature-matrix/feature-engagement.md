# Funktionsinteraktionsanalyse

Erfahren Sie, wie Sie die **Funktionsinteraktionsansicht** in [Adobe Product Analytics](../../adobe-product-analytics/adobe-product-analytics-overview.md). Diese Analyse bietet Einblicke darüber, wie oft Ihre Produktfunktionen verwendet werden und wie viele Personen sie verwenden. Vergleichen Sie mehrere Funktionen gleichzeitig, um optimale Vorteile zu erzielen. Dies hilft Ihnen bei Ihren Investitionsentscheidungen, indem Sie Ihre Kernfunktionen, Leistungsmerkmale, einmaligen und fragwürdigen Funktionen verstehen.

Weitere Informationen dazu finden Sie in der [Dokumentation](https://experienceleague.adobe.com/en/docs/analytics-platform/using/guided-analysis/feature-matrix/engagement).
